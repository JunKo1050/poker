テキサスホールデム ルール

進行

各プレイヤーに手札（ホールカード）を2枚配り、場に共有カード（コミュニティ）を最大5枚開きます。

プリフロップ → フロップ(3枚) → ターン(1枚) → リバー(1枚) の順に、各段階でベットを行います。

アクション

🚪 フォールド：降りる　／　✅ チェック：賭けずに次へ（誰も賭けていない時）

📞 コール：相手の賭けに合わせる　／　⬆️ レイズ：賭け額を上げる

役の強さ（弱い→強い）

ハイカード → ワンペア → ツーペア → スリーカード → ストレート → フラッシュ → フルハウス → フォーカード → ストレートフラッシュ（→ ロイヤル）

チップ・ブラインド

初期スタック 30,000点（1BB=100点）。SB50 / BB100 から始まり、20ハンドごとにブラインドが2倍になります。

チップが足りない場合はオールイン（持ち分だけ参加、超過分はサイドポット）。チップが0になると脱落します。

操作

キーボード: F=フォールド ／ C=チェック・コール ／ R=レイズ ／ ← →=レイズ額の微調整（↑ ↓で大きく変更）。

他のプレイヤーの番のとき、右下の「予約：チェック/フォールド」を押しておくと、自分の番が来た瞬間に自動でチェック（無料のとき）またはフォールドします。

右上の ⏩ ボタンでゲーム速度（×1 / ×1.5 / ×2）を切り替えられます。
